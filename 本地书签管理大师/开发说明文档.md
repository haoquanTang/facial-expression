
# 项目：本地书签管理大师

## 目标

开发一个纯前端的增强型书签管理工具，解决浏览器原生书签管理混乱的问题。

## 核心功能

1. 书签管理：
   - 添加书签（名称、URL、备注、标签）
   - 编辑/删除书签
   - 按标签分类（支持多标签）
2. 搜索与排序：
   - 全文搜索（匹配名称、URL、备注）
   - 排序方式（按添加时间/访问频率/名称）
3. 数据管理：
   - 导入浏览器书签（解析JSON格式）
   - 导出为JSON文件备份
   - 本地存储（使用IndexedDB，避免数据丢失）
4. 界面功能：
   - 标签云展示（点击标签筛选对应书签）
   - 拖拽调整书签顺序

## 技术栈

- 前端：HTML、JavaScript（原生）、Tailwind CSS
- 存储：IndexedDB（或localStorage）
- 工具：原生File API（处理导入导出）

## 输出要求

1. 单页应用代码（HTML+CSS+JS）
2. 离线可用，无需后端
3. 包含导入浏览器书签的操作指南
4. 代码结构清晰，分模块（存储模块、UI模块、工具函数）
